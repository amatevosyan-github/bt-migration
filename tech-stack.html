Технический стек

Полный список технологий, используемых в проекте.

Core Фреймворки

Vue 3.4+

  • Зачем: Современный прогрессивный фреймворк
  • Особенности:
    • Composition API
    • Reactivity система
    • Отличная производительность
    • TypeScript support
    • Большая экосистема

Vite 5+

  • Зачем: Быстрый build tool
  • Особенности:
    • Мгновенный HMR (Hot Module Replacement)
    • Быстрая сборка в production
    • ES modules native support
    • Отличная DX (Developer Experience)
    • Plugin система

Vuetify 3.5+

  • Зачем: Material Design UI framework
  • Особенности:
    • 100+ готовых компонентов
    • Responsive система
    • Theme customization
    • Dark mode из коробки
    • Accessibility
    • TypeScript support

State Management и Routing

Vue Router 4

  • Зачем: Official router для Vue 3
  • Особенности:
    • Composition API support
    • Navigation guards
    • Lazy loading
    • Nested routes
    • Dynamic routing

Pinia 2

  • Зачем: Official state management для Vue 3
  • Особенности:
    • Composition API style
    • TypeScript support из коробки
    • Modular по дизайну
    • DevTools integration
    • Легкая и простая API
    • Hot module replacement

HTTP и Real-time

Axios 1.6+

  • Зачем: HTTP client
  • Особенности:
    • Interceptors
    • Request/Response transformation
    • Automatic JSON transformation
    • Cancel requests
    • Timeout support
    • CSRF protection

Socket.io Client 4+

  • Зачем: WebSocket библиотека
  • Особенности:
    • Автоматическое переподключение
    • Room/Namespace support
    • Fallback транспорты
    • Binary support
    • Acknowledgements

Альтернатива: Native WebSocket API

Утилиты и Библиотеки

VueUse

  • Зачем: Collection of Vue Composition utilities
  • Особенности:
    • 200+ composables
    • Tree-shakeable
    • TypeScript support
    • SSR friendly

Примеры: useDark, useLocalStorage, useFetch, useDebounce

BigNumber.js

  • Зачем: Точные финансовые вычисления
  • Особенности:
    • Arbitrary precision
    • Decimal arithmetic
    • Избегание floating point ошибок

Day.js

  • Зачем: Работа с датами
  • Особенности:
    • Легкая (2KB)
    • API совместимый с Moment.js
    • Immutable
    • Plugin система
    • i18n support

Chart.js / ECharts

  • Зачем: Графики и визуализации
  • Выбор:
    • Chart.js - простой, легкий
    • ECharts - мощный, много опций

TradingView Lightweight Charts

  • Зачем: Торговые графики
  • Особенности:
    • Оптимизирован для финансовых данных
    • Candlestick charts
    • Real-time updates
    • Производительность

Yup / Vuelidate

  • Зачем: Валидация форм
  • Выбор:
    • Yup - Schema-based validation
    • Vuelidate - Vue-native validation

Интернационализация

Vue I18n 9

  • Зачем: Мультиязычность
  • Особенности:
    • Composition API support
    • Pluralization
    • Number formatting
    • Date/Time formatting
    • Lazy loading локалей
    • TypeScript support

Dev Tools

TypeScript (опционально, но рекомендуется)

  • Зачем: Type safety
  • Особенности:
    • Static typing
    • Better IDE support
    • Refactoring tools
    • Catch errors на этапе разработки

Vitest

  • Зачем: Unit тестирование
  • Особенности:
    • Vite-native
    • Jest-compatible API
    • Быстрый
    • Watch mode
    • Coverage reports

Playwright

  • Зачем: E2E тестирование
  • Особенности:
    • Multi-browser support
    • Auto-wait
    • Network interception
    • Screenshot/Video
    • Debug mode

ESLint

  • Зачем: Code linting
  • Особенности:
    • Enforce coding standards
    • Catch potential bugs
    • Pluggable
    • Auto-fix

Prettier

  • Зачем: Code formatting
  • Особенности:
    • Opinionated
    • Consistent style
    • Auto-format
    • Editor integration

Husky

  • Зачем: Git hooks
  • Особенности:
    • Pre-commit hooks
    • Pre-push hooks
    • Enforce quality checks

Commitlint (опционально)

  • Зачем: Валидация commit messages
  • Особенности:
    • Enforce conventions
    • Semantic commits
    • Changelog generation

Deployment

Firebase Hosting (временно для демо)

  • Зачем: Быстрый hosting для прототипов
  • Особенности:
    • Бесплатный tier
    • CDN
    • SSL из коробки
    • Preview channels
    • Easy deployment

Docker

  • Зачем: Контейнеризация для production
  • Особенности:
    • Consistent environments
    • Easy deployment
    • Scalability
    • Portability

Nginx

  • Зачем: Web server для production
  • Особенности:
    • High performance
    • Reverse proxy
    • Gzip compression
    • Caching
    • Load balancing

Monitoring и Analytics

Sentry

  • Зачем: Error tracking
  • Особенности:
    • Real-time alerts
    • Stack traces
    • Breadcrumbs
    • Release tracking
    • Performance monitoring

Google Analytics / Amplitude

  • Зачем: User analytics
  • Особенности:
    • User tracking
    • Event tracking
    • Funnel analysis
    • Retention metrics

Hotjar (опционально)

  • Зачем: User behavior
  • Особенности:
    • Heatmaps
    • Session recordings
    • Feedback
    • Surveys

CI/CD

GitHub Actions

  • Зачем: CI/CD automation
  • Особенности:
    • Native GitHub integration
    • Matrix builds
    • Secrets management
    • Marketplace actions

Преимущества выбранного стека

Производительность

  • ⚡ Vite обеспечивает мгновенный HMR
  • ⚡ Vue 3 оптимизирован и быстрее Vue 2/React
  • ⚡ Vuetify 3 легче предыдущих версий
  • ⚡ Tree-shaking уменьшает bundle size

Developer Experience

  • 🎯 Composition API проще и понятнее
  • 🎯 TypeScript support из коробки
  • 🎯 Отличные DevTools
  • 🎯 Hot Module Replacement

Maintainability

  • 🔧 Модульная архитектура
  • 🔧 Единый стек технологий
  • 🔧 Modern best practices
  • 🔧 Хорошая документация

Scalability

  • 📈 Pinia для масштабируемого state
  • 📈 Модульная структура
  • 📈 Lazy loading
  • 📈 Code splitting

Ecosystem

  • 🌐 Огромная экосистема Vue
  • 🌐 Vuetify компоненты
  • 🌐 VueUse utilities
  • 🌐 Множество плагинов
