Фаза 8: Интернационализация

Длительность: 1-2 недели

Цели фазы

Добавить поддержку нескольких языков (английский и русский) с помощью Vue I18n.

8.1 Настройка Vue I18n

Задачи:

  • [ ] Установить Vue I18n
  • [ ] Настроить конфигурацию i18n
  • [ ] Интегрировать i18n в приложение

8.2 Миграция переводов

Структура файлов переводов

Задачи:

  • [ ] Создать структурированные файлы переводов из всех репозиториев
  • [ ] Создать en.json с переводами на английский
  • [ ] Создать ru.json с переводами на русский

Извлечение переводов из существующих проектов

Задачи:

  • [ ] Извлечь переводы из exchange-website (Nuxt 2)
  • [ ] Извлечь переводы из next-bit.team-react-front (Next.js)
  • [ ] Извлечь переводы из bit-team-auth-front (React)
  • [ ] Объединить все переводы в единую структуру
  • [ ] Убедиться в полном покрытии всех строк

8.3 Composable для i18n

Задачи:

  • [ ] Создать useLocale composable для работы с локалями
  • [ ] Реализовать функции для форматирования валюты, чисел и дат
  • [ ] Создать LanguageSelector компонент для переключения языка

8.4 Lazy loading локалей

Задачи:

  • [ ] Реализовать ленивую загрузку локалей для оптимизации
  • [ ] Загружать только дефолтную локаль при инициализации
  • [ ] Создать функцию для динамической загрузки других локалей

Deliverables фазы 8

По завершению фазы 8 должны быть готовы:

  1. ✅ Vue I18n - настроен и интегрирован
  2. ✅ Файлы переводов - en.json и ru.json с полным покрытием
  3. ✅ useLocale composable - удобная работа с локалями
  4. ✅ LanguageSelector - компонент переключения языка
  5. ✅ Все компоненты - использование i18n

Чеклист завершения фазы 8

  • [ ] Vue I18n установлен и настроен
  • [ ] Файлы переводов созданы (en.json, ru.json)
  • [ ] Все строки переведены на оба языка
  • [ ] useLocale composable создан
  • [ ] LanguageSelector компонент создан
  • [ ] Все компоненты используют i18n
  • [ ] Переключение языка работает
  • [ ] Форматирование чисел и дат работает правильно
  • [ ] Язык сохраняется между сессиями
