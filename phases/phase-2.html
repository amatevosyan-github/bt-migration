Фаза 2: Дизайн-система и UI Kit

Длительность: 3-4 недели

Цели фазы

Создать единую дизайн-систему на базе Vuetify 3 и Figma, построить UI Kit с компонентами по принципу Atomic Design.

Важность фазы

UI Kit - это фундамент всего приложения. Качественно сделанный UI Kit на этом этапе ускорит разработку всех модулей в 2-3 раза.
