🎯 MVP 1: Базовая функциональность + Auth

Цель: Работающая система авторизации с современным UI

Длительность: 8-10 недель

Включенные фазы

  • ✅ Фаза 0: Подготовка и исследование (1-2 недели)
  • ✅ Фаза 1: Инициализация проекта (1 неделя)
  • ✅ Фаза 2: Дизайн-система и UI Kit (3-4 недели)
  • ✅ Фаза 3: Core функциональность (2-3 недели)
  • ✅ Фаза 4: Миграция Auth модуля (2-3 недели)

Deliverables

  • [x] Проект инициализирован (Vue 3 + Vite + Vuetify)
  • [x] UI Kit создан (30-50 компонентов)
  • [x] Темы настроены (Dark/Light)
  • [x] Auth модуль полностью работает:
    • Login/Register
    • Email verification
    • Password reset
    • 2FA
    • OAuth
  • [x] Core инфраструктура:
    • Routing
    • State management (Pinia)
    • API layer
    • WebSocket plugin
  • [x] Firebase Hosting для демо

Критерии успеха

  • ✅ Можно зарегистрироваться и войти
  • ✅ Email verification работает
  • ✅ 2FA можно включить/отключить
  • ✅ OAuth работает (Google, etc.)
  • ✅ Темная/светлая тема переключается
  • ✅ UI выглядит современно и согласованно

  • [x] Create Offer страница
  • [x] Trade страница (полный цикл сделки)

Критерии успеха

  • ✅ Можно создать объявление

    2. Криптовалюта блокируется в escrow

    5. Криптовалюта переводится buyer

  • ✅ Можно открыть спор
  • ✅ Можно оставить отзыв

Заключение

Этот roadmap предоставляет четкий план миграции с измеримыми milestone'ами. Каждый MVP этап приносит реальную ценность и может быть задеплоен для тестирования.

Гибкость: План может быть адаптирован на основе feedback'а и изменения приоритетов.

Коммуникация: Регулярные обновления progress'а и демо на каждом этапе.

Качество: Акцент на качество кода, тестирование и документацию на каждом этапе.
